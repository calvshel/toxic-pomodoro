# Toxic Pomodoro

Toxic Pomodoro is a tiny standard-library Python timer that counts down a classic 25-minute work sprint and punishes early abandonment with a sarcastic red roast.

## Usage

```bash
python pomodoro.py
```

## Features

- 25-minute countdown with a single updating terminal line
- Pure Python standard library only
- Interrupt handling for `Ctrl+C`
- Randomly chosen anti-quit roast in aggressive red terminal styling

## Notes

- The script is intentionally rude. Use it only if you enjoy hostile productivity theatre.
- The timer is designed to be simple, portable, and easy to run without dependencies.
